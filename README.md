# eSportBot

Discord Bot for League and eSport-related content.

# Usage

 ##### _1. Game spectator:_
 
 ##### _Description:_
Shows the participants of the match for a given summoner.
Information includes:
- Team sides
- Picked champions
- Elo
- Win-Lose ratio
##### _Command:_
`/gameinfo <summoner-name>`
